# CanvasFlow 低保真原型

## 桌面布局

```text
┌ 顶部：工作流名称 / 撤销重做 / 校验 / 保存 / 试运行 ┐
├ 节点库 ┤             无限画布 / 小地图             ├ 配置面板 ┤
```

目标尺寸：1366x768 与 1920x1080。窄屏收起节点库和配置面板，保留画布查看与基础键盘操作。
